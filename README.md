# claude
# HR Document Assistant

## 소개
Claude API를 활용한 HR 문서 자동화 및 한국 노동법 컴플라이언스 보조 도구입니다.
인사 담당자의 반복적인 문서 작업을 줄이고, 노동법 기준에 맞는 문서 작성을 지원합니다.

## 주요 기능
- 근로계약서 초안 자동 생성
- 한국 노동법 기반 급여/근태 검토 보조
- HR 공문 및 사내 공지 문서 작성 지원
- 퇴직면담 보고서 자동화

## 사용 기술
- Anthropic Claude API
- Python / JavaScript

## 활용 대상
- 중소기업 인사 담당자
- HR 업무 자동화가 필요한 팀

## 사용 목적
반복적인 HR 문서 업무를 자동화하여 인사 담당자가
더 중요한 업무에 집중할 수 있도록 돕는 오픈소스 프로젝트입니다.
